**M-Tweaker** is an PC optimization platform for Windows. Start with the **Network Optimizer** — deploy 30+ OS-level network tweaks, benchmark latency, and manage per-game QoS — then watch as new products are added over time.

> Built with **Tauri 2 + React + TypeScript + Rust**.  

---

## Current Product

### Network Optimizer

Deploy, verify, and rollback 30+ OS-level network tweaks designed to reduce ping, eliminate jitter, and maximize throughput.

- **One-click deploy** — apply all recommended tweaks in seconds
- **Automatic backups** — rollback any tweak instantly
- **Real-time verification** — confirms each tweak is active
- **Benchmarking** — measure ping and jitter before and after
- **Per-game QoS** — DSCP priority tagging per executable
- **Profiles** — Competitive Gaming and Stable Work presets

---

## What's Coming

M-Tweaker is a growing product hub — the Network Optimizer is just the beginning. New optimization tools will be added over time, each expanding what the platform can do.

## Download

Grab the latest installer from [Releases](https://github.com/M-TWEAKERS/mtweaker/releases).

### Requirements

- **OS:** Windows 10 (build 10240+) or Windows 11
- **RAM:** 512 MB minimum
- **Storage:** ~100 MB
- **Admin rights** required for applying tweaks

---

## Quick Start

1. Download the latest `.exe` from [Releases](https://github.com/yourusername/m-tweaker/releases)
2. Install and launch
3. Enter your license key
4. Navigate to the **Network Optimizer**
5. Hit **ANALYZE** to scan your system
6. Hit **APPLY ALL** to deploy every recommended tweak
